This project involved creating a fully functional e-commerce platform using modern web technologies. The platform allows users to browse products, add items to a shopping cart, and securely complete transactions.

Steps Taken to Complete the Project:

  1. Developed the front-end using React.js
  2. Integrated a secure payment gateway
  3. Implemented a dynamic product catalog with filters
  4. Developed a user authentication system for customers
  5. Tested the platform and deployed it on AWS
