This project involved designing and developing a responsive personal portfolio website to showcase my skills, projects, and achievements. It includes a clean, minimalist design with sections for biography, portfolio, and contact details.

Steps Taken to Complete the Project:

  1. Designed wireframes and layout using Figma
  2. Coded the front-end using HTML, CSS, and JavaScript
  3. Integrated a contact form with email functionality
  4. Optimized for mobile responsiveness
  5. Launched the website on a custom domain
